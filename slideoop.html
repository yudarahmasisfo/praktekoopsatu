Contoh Class dalam PHP

<?php
class Mobil {
  // Property
  public $merk;
  public $warna;
  private $kecepatan = 0;

  // Constructor
  public function __construct($merk, $warna) {
    $this->merk = $merk;
    $this->warna = $warna;
  }

  // Method
  public function jalan() {
    $this->kecepatan = 50;
    return "Mobil sedang berjalan";
  }
}
?>

Membuat Objek dari Class

<?php
// Membuat objek dari class Mobil
$mobil1 = new Mobil("Toyota", "Merah");
$mobil2 = new Mobil("Honda", "Biru");

// Menggunakan method
echo $mobil1->jalan();
?>

Coba Sendiri!

Property, Method & Access Modifier

🏷️ Property

Variabel yang menyimpan data objek

public $nama;
private $umur;

⚙️ Method

Function yang mendefinisikan behavior objek

public function getName() {
  return $this->nama;
}

🔒 Access Modifier

Mengatur tingkat akses property/method

  • public: Dapat diakses dari mana saja
  • private: Hanya dalam class itu sendiri
  • protected: Class dan turunannya

Live Demo: Class Buku

Input Data Buku

class Buku {
  public $judul;
  public $penulis;
  public $tahun;

  public function __construct($j, $p, $t) {
    $this->judul = $j;
    $this->penulis = $p;
    $this->tahun = $t;
  }

  public function getInfo() {
    return $this->judul . " oleh " . $this->penulis;
  }
}

Keuntungan Pemrograman Berorientasi Objek

🔄

Reusability

Code dapat digunakan kembali melalui inheritance dan composition, mengurangi duplikasi code.

🛡️

Encapsulation

Data dan method dibungkus dalam class, memberikan keamanan dan kontrol akses yang lebih baik.

🔧

Maintainability

Code lebih mudah dipelihara dan dimodifikasi karena struktur yang terorganisir dengan baik.

📈

Scalability

Aplikasi dapat dikembangkan dengan mudah melalui penambahan class dan fitur baru.

🎯

Modularity

Aplikasi terbagi menjadi modul-modul kecil yang independen dan mudah dikelola.

👥

Collaboration

Tim developer dapat bekerja secara paralel pada class yang berbeda tanpa konflik.

Latihan: Buat Class Mahasiswa

Instruksi:

Buat class Mahasiswa dengan:

  • Property: nama, nim, jurusan
  • Constructor untuk inisialisasi
  • Method getInfo() yang mengembalikan informasi lengkap
  • Method setNilai() dan getNilai() untuk nilai
